02 · Motivation

왜 만들었나요?

교사의 실제 업무 현장에서 느낀 불편함에서 출발했습니다.

📝

공문서 작성의 번거로움

예산 공문 작성 시 숫자를 한글로 변환하는 작업이 매번 반복됩니다. 인터넷 검색이나 별도 프로그램 없이 즉시 변환이 필요했습니다.

💰

천원 단위 표기 혼란

예산서에 자주 사용하는 천원 단위 표기(1,234천원)를 계산기에서 바로 얻을 수 없어 별도로 계산해야 했습니다.

⌨️

빠른 실행의 필요성

문서 작업 중 계산기가 필요할 때 마다 마우스를 움직여 실행하는 과정이 번거로웠습니다. 키보드만으로 즉시 실행되길 원했습니다.

🤖

AI 도구 활용 역량 함양

코드를 직접 작성하지 않고 Claude Code와 대화만으로 실제 업무 도구를 만들 수 있다는 것을 직접 경험하고 동료 교사들과 공유하기 위해 제작했습니다.

03 · How It's Made

핵심 프롬프트

코드 한 줄 작성하지 않았습니다. Claude Code에 요구사항을 말하면 AI가 코드를 작성했습니다. 아래는 주요 기능을 만든 실제 프롬프트입니다.

기능 요청 1 — 초기 설계

계산기 프로그램을 만들고 싶다. 창을 줄이고 늘림에 따라 버튼과 숫자가 자동으로 변화되길 원한다. NumLock 버튼을 2번 연속으로 누르면 자동 실행되게 하고, 계산된 숫자를 공문서 한글 금액으로 자동 변환해 아래에 표시하고 클릭하면 복사되게 해라.
구현 결과 반응형 계산기 UI, NumLock 더블클릭 실행, 한글 금액 자동 표시 및 클립보드 복사

기능 요청 2 — 오류 수정

천원 단위 변환할 때 1,234천원이 되어야 하는데 1.234로 변환되고 있다. 수정하라. 그리고 NumLock을 또 2번 누르면 계산기가 또 실행되게 하고, X를 눌렀을 때 완전 종료되게 하라.
구현 결과 천원 단위 소수점 버그 수정, 다중 인스턴스 지원, X 버튼 동작 수정

기능 요청 3 — 설정 및 만든이

설정 버튼을 만들어서 X 동작, 항상 위에 표시, 자동 시작, 투명도 등을 조절하게 하라. 만든이는 알잘딱버튼을 참고해 팝업으로 구현하고, 체크박스는 색깔 없는 박스로 구현하라.
구현 결과 설정 다이얼로그, 커스텀 다크 체크박스

기능 요청 4 — 소수점 표시 개선

나눗셈 결과의 소수점 10자리가 전부 표시되어 정수 부분이 안 보인다. 창 너비에 따라 소수점 자릿수를 조절하고, 잘린 자리는 말줄임표(...)로 표시하라. 최소 소수점 2자리는 표시하라.
구현 결과 창 너비 연동 동적 소수점 표시 (260px→2자리, 660px→10자리), 말줄임 처리

04 · Use Cases

활용 방법 및 사례

교사의 실제 업무 흐름에 맞춰 설계된 기능들입니다.

1

예산 공문서 작성

예산액을 계산기에 입력하고 [한글 변환] 버튼을 누르면 즉시 공문서 표준 한글 금액이 클립보드에 복사됩니다. 문서에 바로 붙여넣기 할 수 있습니다.

예) 1,234,567 → 금 일백이십삼만사천오백육십칠원 정
2

예산서 천원 단위 표기

학교 예산서나 품의서 작성 시 [천원] 버튼으로 천원 단위를 즉시 변환합니다.

예) 1,234,567 → 1,234천원
3

문서 작업 중 빠른 계산기 호출

한글, MS Word 등 문서 작업 중 키보드에서 손을 떼지 않고 NumLock 키를 1초 이내에 2번 누르면 계산기 창이 바로 열립니다. 계산 후 X를 누르면 트레이로 숨겨지고, 다시 NumLock 2번으로 새 창을 열 수 있습니다.

4

여러 계산기를 동시에 사용

항목별로 계산기를 여러 개 띄워서 각 창에서 별도로 계산할 수 있습니다. NumLock 더블클릭 시마다 새 계산기 창이 하나씩 추가됩니다.

프로그램 소개

교사 업무에 최적화된 계산기의 주요 기능을 소개합니다.

공문서 한글 금액 변환

= 또는 [한글 변환] 버튼으로 공문서 표준 "금 일백이십삼만원 정" 형식 자동 변환. 클릭 시 클립보드 복사.

[천원] 버튼으로 "1,234천원" 형식 즉시 변환. 클릭 시 클립보드 복사.

⌨️

NumLock 단축 실행

어떤 프로그램을 쓰는 중에도 NumLock 2번으로 새 계산기 창 실행. 여러 창 동시 사용 가능.

📐

반응형 UI

창 크기에 따라 버튼, 숫자, 폰트가 자동 조절됩니다. 나눗셈 소수점도 창 너비에 맞게 동적 표시.

🔔

시스템 트레이 상주

X 버튼 클릭 시 트레이로 숨겨져 백그라운드 대기. 우클릭으로 열기/종료 가능.

⚙️

다양한 설정

항상 위에 표시, 창 투명도 조절, Windows 시작 시 자동 실행, X 버튼 동작 선택.

🖥️

단일 실행 파일

설치 불필요. .exe 파일 하나를 복사해서 바로 실행. Windows 10/11 지원.

🤖

AI로 만든 도구

Claude Code와 대화만으로 제작. 코드 한 줄 없이 교사가 직접 기획하고 완성한 실용 도구.

06 · QnA

자주 묻는 질문

사용 중 궁금한 점을 확인하세요.

설치 없이 교사용계산기.exe 파일 하나만 실행하면 됩니다. 바탕화면이나 원하는 폴더에 복사해서 사용하세요. Windows 10 / 11 에서 동작합니다.
NumLock 키를 1초 이내에 2번 연속으로 눌러야 합니다. 계산기가 이미 여러 개 실행 중일 경우 1.5초 이내에는 중복 실행이 방지되므로, 잠깐 기다렸다가 다시 눌러보세요.
기본 설정이 트레이 숨김으로 되어 있습니다. 우측 하단 트레이 아이콘을 우클릭하여 "종료"를 선택하거나, 설정(⚙) → "X 버튼: 백그라운드 대기" 체크를 해제하면 X 클릭 시 완전 종료됩니다.
본 프로그램은 공문서 표준(행정안전부 기준)에 따라 모든 자릿수에 "일"을 포함합니다. 예) 123 → "일백이십삼" (백이십삼 ✗). 소수점 이하는 반올림 후 정수로 변환합니다.
설정(⚙) 버튼 → "Windows 시작 시 자동 실행" 체크 후 저장을 누르면 됩니다. 단, 프로그램 파일을 나중에 다른 폴더로 이동하면 자동 실행 경로가 바뀌므로 다시 설정해 주세요.
현재 버전은 Windows 전용입니다. NumLock 감지와 자동 시작 기능이 Windows API에 의존합니다. 소스코드(GitHub)를 받아 Python 환경에서 직접 실행하면 macOS에서도 일부 기능은 동작할 수 있습니다.
